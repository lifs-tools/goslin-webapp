News and Updates

  • v1.0.19 - May 12th, 2020 

    This release adds support for adducts, currently limited to those supported by LipidCreator. Thus, any lipid shorthand notation name can now be combined with an adduct. Please note that adduct charges need to be explicit, thus '[M+H]+' needs to be provided as '[M+H]1+'.

Citing Goslin

  1. Goslin - A Grammar of Succinct Lipid Nomenclature. Dominik Kopczynski, Nils Hoffmann, Bing Peng, Robert Ahrends. 2020 (BioRxiv Preprint)
